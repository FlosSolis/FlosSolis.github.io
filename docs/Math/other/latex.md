# LaTeX 常用符号整理

> 记录在数学学习过程中所用到的LaTeX 符号

## 运算

### 极限运算

| Type   | Typeset |
| :----: | :-----: |
| `\lim` | $\lim$  |
|`\lim\limits_{x\to∞}`|$\lim\limits_{x\to∞}$|

### 复杂运算

|                Type                 |               Typeset               |
| :---------------------------------: | :---------------------------------: |
|            `\sqrt{abc}`             |            $\sqrt{abc}$             |
|           `\sqrt[n]{abc}`           |           $\sqrt[n]{abc}$           |
|          `\frac{abc}{xyz}`          |          $\frac{abc}{xyz}$          |
|           `\int_{a}^{b}`            |           $\int_{a}^{b}$            |
| `\frac{\mathrm{d} y}{\mathrm{d} x}` | $\frac{\mathrm{d} y}{\mathrm{d} x}$ |
|          `\sum_{i=1}^{n}`           |          $\sum_{i=1}^{n}$           |

## 符号



|   Type    | Typeset  |                           Comment                            |
| :-------: | :------: | :----------------------------------------------------------: |
| ` \infty` | $\infty$ |                                                              |
| `\ldots`  | $\ldots$ | `\ldots `用于表示与文本的基线对齐的省略号。<br>它通常用于横向排列的元素之间: 如列表中的项或加、减运算符之间的省略。 |
| `\cdots`  | $\cdots$ | `\cdots `用于表示与数学表达式中的操作符垂直居中的省略号。<br>它通常用于乘法、除法等操作符之间的省略。 |
|           |          |                                                              |
