# 英语作文模板

## 四六级



## 考研—小作文

### 首段—表达写信的目的：

1. 次要目的（客套礼节）+主要目的（题干重点描写要求的部分）<br/>
   Dear<font color = red> Xx,（首字母都要大写，结尾是逗号，）</font><br/>
   	I am writing this<font color = blue> letter </font>to... / This <font color = blue>notice </font>is to...<br/>
   亲爱的<font color = red>xx</font>，我写<font color = blue>这封信</font>是为了/<font color = blue>本公告</font>是为了<br/>
   次要目的（感谢、祝贺、道歉等纯粹情感性的目的，未必会有）：<br/>
   thank you for.../congratulate you for.../apologize for.../do sth, and to<br/>
   <font color = blue> 为…而感谢你/祝贺你/向你道歉/其他目的，</font>并且<br/>
2. 主要目的（提建议、意见，介绍信息，进行推荐...文章的核心主旨）：<br/>
   ①offer you some suggestions/information about sth/how to do sth.<br/>
   给出关于<font color = blue> 某事/如何做某事</font>的<font color = blue> 建议/信息</font><br/>
   ② extend you an invitation to do sth.<br/>
   <font color = blue> 邀请</font>你做某事<br/>
   ③ recommend you sth<br/>
   <font color = blue> 推荐</font>给你某事物<br/>
   ④ do sth<br/>
   做某事

### 尾段—表达对读信人的期望：

按是否需要对方回信分为以下两种（部分题目回不回信都可以）

1. 否：（公告、招募、问询…）→没有需求，创造需求<br/>
   
    I would be glad if you found the <font color = blue> information/suggestions</font> mentioned above helpful.
    Should you have any further questions, please feel free to contact me.<br/>
    
    <p align="right" >Yours truly,<br> Li Ming</p> 
    
    希望以上<font color = blue> 信息/建议</font>对你有用。如果有其它问题，请随时与我联系。
    
    <p align="right" >你诚挚的，<br/>李明</p>
    
    
    
  2. 是：（建议、介绍、推荐、感谢、道歉、邀请、投诉、申请…）→期待回复<br/>
      I would appreciate it if you could take my <font color = blue> complaint / application / resignation /recommendation / invitation / questions</font> into serious consideration.
      And I would be much obliged to you for an early and positive reply.<br/>
      
      <p align="right" >Yours truly,<br> Li Ming</p> 
      
      希望你能重视或认真考虑我的<font color = blue> 投诉/申请/辞职/建议/邀请/问题</font>。如能早日得到你的积极回复，我将不胜感谢。<br/>

<p align="right" >你诚挚的，<br/>李明</p>

### 第二段—主体段

#### 整体架构:

Regarding... , there are several points I would like to mention.<br/>
First of all, 论点1.<br/>
What’s more, 论点2.<br/>
In addition, 论点3.<br/>
For more details, you could refer to the attachment below. （28 词）

#### 具体论点:

三元：是什么（说明介绍）、为什么（原因利弊）、怎么办（建议意见）

1. <font color =red>是什么（以说明介绍活动信息为主）：</font>
   - <font color =red>时间、地点：</font> <br/><font color =blue>
    他将于下周日在主楼举行/它位于主楼里（旁边）/我们下周日在主楼集合并前往目的地（请保证届时你有时间参加）</font><br/>
     <font color =blue> it will be held / it is located</font> in (beside) the Main Building (next Sunday)/ we will gather in the Main Building next Sunday and set off to the spot (, and please make sure you are available then).
    - <font color = red>出席人物：</font><br/>
       我们的工作人员届时将会为你提供一些必要的<font color = blue>培训/介绍/引导</font> <br/>
      our staff will be there to provide you (with) necessary <font color = blue>orientation/guidance</font>.
    - <font color = red>着装要求：</font> <br/>
    出席活动时，着装应该<font color = blue>得体/暖和/朴素</font><br/>
       it is highly recommended to be<font color = blue> properly/warmly/plainly
       </font> dressed for the occasion.
2. <font color =red>为什么（分析利弊）:</font>
    - <font color =red> 权威认可：</font>
        - ta获得了权威机构的官方<font color =blue>奖项/资质/认可</font>...<br/>
        have (has) an official <font color = blue>award / qualification / recognition</font> from the authority.
        - 被授予联合国<font color = blue>物质/非物质文化</font>遗产称号。<br/>
        are (is) awarded with the title of <font color = blue>Material / Intangible Cultural Heritage</font> from the United Nations.
    - <font color =red>群众基础：</font><br/>
      <font color =orange>人民群众/很多社交媒体平台</font>对ta<font color =blue>很欢迎/评价很高/很关注担心</font><br/>
      enjoy (enjoys) a widespread <font color =blue>popularity / reputation / concern</font><font color =orange> among 人民群众/on multiple social platforms</font>.
    -  <font color = red>个人喜欢：</font><br/>我个人觉得xx很有<font color = blue>吸引力/有趣/有益/必要</font><br/>
      I personally find it <font color= blue>attractive / interesting / beneficial / necessary</font> to do sth.
3. <font color = red>怎么办（建议意见）：</font>
- <font color = red>端正认识：</font><br/>
      端正认识，并建立起关于xx的正确价值观是首要任务<br/>
      it’s advisable to put things into perspective and establish correct values about sth primarily.
    - <font color = red>实践经验：</font><br/>
      你可以在知乎社区上寻找该领域的<font color = blue>相关/先进/国际/历史</font>经验<br/>
      you could look for <font color = blue>relevant / advanced / international / historical</font> experience in this aspect on Zhihu Community.
    - <font color = red>知行合一：</font><br/>
      应该及时制定合理、可行的计划并优先执行<br/>
      a practical and reasonable plan should be promptly carried out and remain high on your agenda.

主段（三个方面）：

- 是什么（以说明介绍活动信息为主）：<br/>首先，<font color =blue>他将于下周日在主楼举行/它位于主楼里（旁边）/我们下周日在主楼集合并前往目的地（请保证届时你有时间参加）</font>。更重要的是，我们的工作人员届时将会为你提供一些必要的<font color = blue>培训/介绍/引导</font>。另外，出席活动时，着装应该<font color = blue>得体/暖和/朴素</font>。欲了解更多详情，您可以参考下面的附件。
- 为什么（分析利弊）：<br/>首先，ta获得了权威机构的官方<font color =blue>奖项/资质/认可</font>。<font color= red size=5>/</font>首先，ta被授予联合国<font color = blue>物质/非物质文化</font>遗产称号。更重要的是，<font color =blue>人民群众/很多社交媒体平台</font>对ta<font color =blue>很欢迎/评价很高/很关注担心</font>。另外，我个人觉得xx很有<font color = blue>吸引力/有趣/有益/必要</font>。欲了解更多详情，您可以参考下面的附件。
- 怎么办（建议意见）：<br/>首先，端正认识，并建立起关于xx的正确价值观是首要任务。更重要的是，你可以在知乎社区上寻找该领域的<font color = blue>相关/先进/国际/历史</font>经验。另外，你应该及时制定合理、可行的计划并优先执行。欲了解更多详情，您可以参考下面的附件。

  <p align="right" >你诚挚的<br/>李明</p>
  
- 是：<br/>希望你能重视或认真考虑我的<font color = blue> 投诉/申请/辞职/建议/邀请/问题</font>。如能早日得到你的积极回复，我将不胜感谢。<br/>

<p align="right" >你诚挚的，<br/>李明</p>
